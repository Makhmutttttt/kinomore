# Kinomore
 Front-end приложение для поиска и просмотра фильмов. Демо: [https://kinomore.netlify.app/](https://kinomore.netlify.app/)
 
# Стек технологий:
  * React + React Hooks
  * TypeScript
  * Redux Toolkit, RTK Query
  * Sass, CSS modules
  * Jest
  * NextJS
  * PWA
  * React-hook-form, Yup
  * Testing Library
  * Storybook
